# 南极狐使用说明

## 快速开始

### 第一步：登录系统

1. 打开南极狐管理面板（通常在 `http://localhost:3000` 或你的Cloudflare域名）
2. 看到标题 **"南极狐"** 和欢迎语 **"欢迎来到南方"**
3. 输入用户名和密码登录

### 第二步：生成UUID

1. 登录后进入操作面板
2. 点击 **"生成UUID"** 按钮
3. 系统会自动生成一个新的UUID
4. 复制并保存这个UUID

### 第三步：配置面板

1. 进入 **"面板设置"**
2. 填入以下信息：
   - **UUID**: 刚才生成的UUID
   - **域名**: 你的Cloudflare绑定域名（如 example.com）
   - **密码**: 设置操作面板密码（d1或KV）
3. 点击 **"保存配置"**

### 第四步：生成节点

1. 进入 **"节点管理"**
2. 选择节点类型：
   - **VLESS** (推荐，最兼容)
   - **TUIC** (性能最好)
   - **Wireguard** (隐私最好)
3. 设置生成数量（1-10个）
4. 点击 **"生成节点"**

### 第五步：获取订阅

1. 在生成的节点页面找到 **"订阅链接"**
2. 可以选择：
   - 复制订阅链接
   - 扫描二维码
   - 导出配置文件

## 功能详解

### 1. 认证系统

#### 注册新用户
```
路径: /api/auth/register
方法: POST
参数: 
  - username: 用户名
  - password: 密码
```

#### 用户登录
- 首次使用需要注册账户
- 使用用户名和密码登录
- 系统会返回访问令牌

### 2. UUID管理

#### 为什么需要UUID？
- UUID是你的唯一身份标识
- 用于生成个性化的代理节点
- 与绑定的域名组合形成最终的节点配置

### 3. 节点生成

#### VLESS节点
- **最推荐**: 兼容所有主流客户端
- **用途**: 通用VPN代理
- **性能**: 良好
- **配置**: 自动化完成

#### TUIC节点
- **特点**: 基于QUIC协议，高效率
- **用途**: 需要性能的用户
- **性能**: 最好
- **要求**: 客户端需支持TUIC协议

#### Wireguard节点
- **特点**: 现代化VPN协议
- **用途**: 注重隐私和安全的用户
- **性能**: 非常好
- **要求**: 客户端需支持Wireguard

## 部署选项

### Cloudflare Pages
静态资源部署

```bash
# 部署前端
npm run build
wrangler pages publish dist
```

### Cloudflare Workers
后端服务部署

```bash
# 部署API
wrangler deploy
```

### Docker本地部署
```bash
docker build -t antarctic-fox .
docker run -p 3000:3000 antarctic-fox
```
